# PDF-Summarizer-Chatbot
PDF Summarizer Chatbot built using Python, Streamlit, and the OpenAI API. The application allows users to upload a PDF document, extract its text, and generate a concise summary using an AI language model.

# Features
Upload PDF files through a simple web interface
Extract text from PDF documents
Generate clear and concise summaries using the OpenAI API
Interactive web UI built with Streamlit
